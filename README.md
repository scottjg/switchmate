# switchmate

it's a commandline utility to control the [Switchmate](http://switchmatehome.com/).

```
$ ./switchmate discover
Scanning for peripherals...
7389F91A-1CC0-464D-894A-C92204D51A49 -46 beacon
$ ./switchmate getauthkey 7389F91A-1CC0-464D-894A-C92204D51A49
Connecting to device...
Connected to Switchmate 7389F91A-1CC0-464D-894A-C92204D51A49, discovering services...
Discovering characteristics...
Got characteristics, Requesting subscription...
Got subscription, writing command...
Acknowledged successful write!
The device is ready to pair, press the button on the switchmate.
vWZWCQ==
$ ./switchmate toggle on 7389F91A-1CC0-464D-894A-C92204D51A49 vWZWCQ==
Connecting to device...
Connected to Switchmate 7389F91A-1CC0-464D-894A-C92204D51A49, discovering services...
Discovering characteristics...
Got characteristics, Requesting subscription...
Got subscription, writing command...
Acknowledged successful write!
Toggled Ok!
```
